# GreenCube Shop 🌿

Premium Hanfstecklinge Shop mit 25 Sorten

## Features
- 🛒 Warenkorb-Funktion
- 🔍 Produktsuche & Filter
- 📱 Responsive Design
- ⚡ Next.js 16 + React 19
- 🎨 Tailwind CSS + shadcn/ui

## Sorten
- AK-47 Simons Cut
- Amnesia Haze
- Blue Dream
- Bubble Gum
- Cali Orange
- Critical Kush
- Gelato
- Gorilla Glue
- OG Kush
- Und 15 weitere...

## Tech Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui

## Installation
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
